# docs

Planning, decisions, and progress for imgthing.

- **[cloudflare-setup.md](./cloudflare-setup.md)** — ⚡ action items you need to do in your
  Cloudflare account (start here).
- **[PROGRESS.md](./PROGRESS.md)** — running log of what's built, per milestone.
- **[imgthing-ui.md](./imgthing-ui.md)** — the UI design language (Bright Studio Glass).
- **[plans/](./plans/)** — the project plan and any milestone-specific plans.
- **[decisions/](./decisions/)** — Architecture Decision Records (ADRs), one file per decision.

## Conventions

- ADRs are numbered `NNNN-short-title.md` and are append-only: to reverse a decision, add a new
  ADR that supersedes the old one (note it in both).
- Keep `PROGRESS.md` current as milestones complete.
